== Import Demo Content ==

1) Download demo content: https://demo.rocksite.pro/genui/demo-content/genuitest.WordPress.xml
2) Log into your site as an administrator.
3) Make sure you activated your new theme before proceeding.
4) Go to Tools → Import
5) Choose 'WordPress' from the list.
6) Upload the demo content using this .xml file which you downloaded in the first step
7) You will be asked to map the authors in this export file to users on the blog. For each author, you may choose to map to an existing user on the blog or to create a new user.
8) You will then have the choice to import attachments, so click on the "Download and import file attachments" box.
9) WordPress will import all the content. You will need assign the menu a theme location & you’ll need to configure widgets. This is a limitation of WordPress import system.
10) Done

== Features ==

= Appearance > Widgets =
Genui has up to 3 widget area. 2 for each sidebar and up to 1 in your footer.

== Menus ==

Genui offers 4 menu areas:

'Main Menu' - main dropdown menu
'Top Menu' - single-line menu at very top
'Footer Menu' - menu in footer area,
'Social Menu' - menu in footer area

== Widgets ==

Genui offers 1 additional widget:

'Extended Recent Post' -  offers you more control when displaying recent posts
